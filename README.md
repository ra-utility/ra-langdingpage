### 上手指南

#### 环境要求
- node: v>=10.0
- vue: >= 2.5.x


#### 安装步骤
```sh
# fork or clone the repository
git clone xxxxx

# install
cd xxx
yarn 

# run
yarn start

```



### 部署

使用 `zeit now` 部署 WEB 端应用([zeit now](https://zeit.co/)).
